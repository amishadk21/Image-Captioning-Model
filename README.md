Image captioning is an machine learning task that involves generating a textual description of an image's content. Unlike image classification, 
which assigns labels to images, or object detection, which identifies and locates objects within images, captioning requires understanding the scene as 
a whole and expressing it in natural language. 

A successful image captioning system must: 
●	Identify objects and their attributes in the image 
●	Understand the relationships between objects 
●	Recognize the context and scene type 
●	Generate grammatically correct and semantically meaningful descriptions
